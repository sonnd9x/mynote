# mynote
This is simple note application using:
- Asp.Net MVC5
- Entity Framework 6
- Bootstrap 3
- Pagedlist

*How to install.
- Clone project and change String connection in Webconfig.
- Publish to  IIS or windows hosting.
*Demo:
- http://mynotes.somee.com/
- admin page: http://mynotes.somee.com//admin (demo | demo)
